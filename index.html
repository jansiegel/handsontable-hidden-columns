Handsontable "hiddenColumns" plugin demo

Feature guide:

The columns property

The columns property defines the columns to be hidden by the plugin. It accepts an array of indexes.
See the all-in-one example for an example usage.

The indicators property

The indicators property defines whether the table should have small triangle shaped indicators in the column headers to indicate the hidden columns. It accepts boolean values.
Default value: true

See the all-in-one example for an example usage.

The copyPasteEnabled property

The copyPasteEnabled property defines whether the table should allow to copy and paste from/into the hidden columns.
Default value: false
